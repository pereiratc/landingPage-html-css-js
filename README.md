# Starter

A clean, minimal front-end starter template for quickly kicking off web projects — no frameworks, no build tools, just pure HTML, CSS, and JavaScript ready to go.

---

## 📁 Project Structure

```
Starter/
├── index.html        # Main entry point
├── css/              # Stylesheets
├── js/               # JavaScript files
└── images/           # Image assets
```

---

## 🛠️ Built With

- **HTML** — Page structure and content
- **CSS** — Styling and layout
- **JavaScript** — Interactivity and behavior

---

## ✨ Features

- Minimal and unopinionated — start building without fighting a framework
- Pre-organized folder structure for assets, styles, and scripts
- No dependencies, no package manager required
- Easy to extend or plug into any workflow

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/pereiratc/Starter.git
   cd Starter
   ```

2. **Open in browser**

   Simply open `index.html` directly in your browser, or use a local dev server:

   ```bash
   # With VS Code Live Server extension, or:
   npx serve .
   ```

3. **Start building!**

   Add your styles to `css/`, your scripts to `js/`, and your images to `images/`.
